Bạn có thể chỉnh lại tham số hình học của Robot như a, rf, re, k ở file delta_pararms.m để động học thuật/nghịch có thể tính chính xác.

Sau khi bấm RUN ở Simmulink thì có thể dùng file DoThi_QuyDao.m để hiể thị đồ thị so sánh giữa mô phỏng và thuật toán.

Có thể lưu lại data sau khi mô phỏng bằng file Save_data.m.
Nếu mở data từ Save_data.m thì dùng file Open_data.m.
Với data được mở từ Open_data.m thì hãy sử dụng Open_file_saved.m để xem đồ thị.
